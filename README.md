## Overview

DeviceDestination is a premium security solutions eCommerce website built with **Next.js**, **TypeScript**, and **Tailwind CSS**.

The platform is designed to deliver a modern online shopping experience for surveillance and security products including:

- CCTV Cameras
- NVR Systems
- Video Door Phones
- Biometric Attendance Systems
- Access Control Solutions
- Networking Products
- Security Accessories

The website combines cinematic visuals, premium branding, responsive design, and SEO-focused architecture while maintaining excellent performance and accessibility.

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/devicedestination.git
```

Navigate into the project

```bash
cd devicedestination
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Run production server

```bash
npm start
```
